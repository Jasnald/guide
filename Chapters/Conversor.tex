\chapter{Data Conversion Utilities}
\label{ch:conversor}

The simulation outputs generated by Abaqus are stored in the proprietary ODB database format. To facilitate high-performance post-processing and open-source visualization (e.g., ParaView), this module provides a pipeline to convert these files into the \textbf{XDMF + HDF5} standard.

The process is divided into three stages: configuration/orchestration, raw data extraction (Python 2.7), and final binary compilation (Python 3.x).
